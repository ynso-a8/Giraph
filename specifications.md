# 🦒 기래프(GIRAFFE) 기능 명세서

**기래프**는 사용자가 매일 자신의 기분을 기록하고 시각화하며, 과거의 나와 미래의 나를 연결하는 감성적인 AI 마인드케어 모바일 웹 서비스입니다. 기분 점수(0~100)에 따라 기린의 목 길이와 표정이 실시간으로 변하는 비주얼 요소가 핵심입니다.

---

### ③ AI 마음 클리닉 (자가진단 & 챗봇)

### ④ 감성 연결소 (과거와 미래의 연결)

| 기능명 | 설명 | 비고 |
| :--- | :--- | :--- |
| **감정 회상 타임캡슐** | 2일 이상 경과한 과거 기록 중 무작위로 고민과 사유를 꺼내어 보며 당시의 감정을 건강하게 회상하도록 돕습니다. | 2일 이상 지난 데이터 필요 |

| **미래의 나에게서 온 AI 위로 편지** | 기록된 일지 개수와 평균 점수를 기 분석하여 '미래의 내가 과거의 나에게 보내주는 응원 편지'를 자동 생성해 줍니다. 기분 상태에 따라 감정 가뭄(우울), 온화한 초원(평온), 기린의 댄스(행복) 등의 문학적 비유가 가미됩니다. | 감성적 가독성 강조 |
| **마음 통계 지수** | 로그 변동성과 회복 시간을 수식화하여 세 가지 수치로 환산해 프로그레스 바로 보여줍니다.<br>• **회복 탄력성**: 우울했던 점수 이후 빠른 평정심 회복 지수.<br>• **마음 안정성**: 감정의 일교차(표준편차)가 적을수록 높은 점수.<br>• **미래 긍정 지수**: 기록 횟수와 시도 횟수가 누적될수록 꾸준히 성장하는 지수. | - |

## 3. 핵심 시스템 디자인 및 아키텍처

1. **Supabase & LocalStorage 하이브리드 아키텍처**
   * [supabase.ts](file:///d:/해달/2026 해달 해커톤/mood-tracker/src/lib/supabase.ts) 모듈에서 API Key 유효성을 감지합니다. Supabase 연동에 실패하거나 환경변수가 제공되지 않은 환경에서는 앱 전체가 LocalStorage 모드(`mood_logs_local`, `giraffe_future_letters`)로 자동 전환(Fallback)되어 중단 없는 서비스 경험을 보장합니다.
2. **반응형 프리미엄 모바일 웹 프레임**
   * [layout.tsx](file:///d:/해달/2026 해달 해커톤/mood-tracker/src/app/layout.tsx)는 PC 브라우저 환경에서도 모바일 기기 규격(`max-w-md`)으로 레이아웃을 집중시켜 뷰포트를 모바일 앱처럼 세련되게 모방합니다.
   * 백그라운드에 은은하게 퍼지는 2개의 유동적인 네온 광원 이펙트(`fixed top-[-10%] blur-[120px]`)가 다크 모드 속에서 감성적이고 몰입감 높은 심미성을 선사합니다.
3. **컴포넌트 중심의 설계**
   * 기린 캐릭터 SVG 컴포넌트인 [GiraffeFace.tsx](file:///d:/해달/2026 해달 해커톤/mood-tracker/src/components/GiraffeFace.tsx)를 핵심 비주얼 노드로 격리하여 슬라이더, 최근 목록 피드, 차트 등의 위젯들이 결합도 낮게 기린의 비주얼 상태를 렌더링하도록 디자인되었습니다.
